Community Profiles Tutorial

Updated January 2022

This tutorial will teach you how to use the interactive map features of the Community Profiles web app. For additional information, or to request a demo, please email Grant Moss (grant.moss@cer-rec.gc.ca).

General Instructions
  • For the best experience, avoid using Internet Explorer.
  • Larger screen sizes (tablet, laptop, desktop) will be easier to use compared to mobile phones.
Basic Map Navigation Laptop/Desktop

While hovering over the map, use your mouse or trackpad to zoom in and out of the map.

Phone/tablet

Use two fingers in a pinch action on the screen to zoom in and out.

In addition to using the zoom options above, the map has + and - buttons that can be clicked to zoom in and out.

The layer button directly below the + and - buttons can be used to filter map layers to make it less cluttered if certain map layers are not required.

It is highly recommended to use the fullscreen option when using the application.

The Reset Map button can be used at anytime during map navigation to re-zoom to the initial starting point. This button will also clear and reset any changes/pop-ups that appear during the use of the app.
